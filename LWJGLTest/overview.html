Implementation of a block-based graphics library with LWJGL!

Examples

First include the Library into your own Project with: Properties, Java Build Path, Libraries, Add JAR. Then the library can already be used with the following code: 

import de.oth.blocklib.*;
public class main {
 public static void main() {
  Configuration config = new Configuration();
  config.setWorldSize(8);
  Game game = new Game(config);
  World world = new World(config.getWorldSize());
  world.setBlock(i, 0, j, BlockType.Stone);
  game.initWorld(world);
  game.start();
 }
}

If there is a need to get in control of the main loop, this can be done by inheriting from the Game class. The world needs to be recreated if there are changes on the world. 

public class ExampleGame extends Game {
  public ExampleGame(Configuration config) {
    super(config);
  }
  public final void update(final float delta) {
    if (!getWorld().isMeshLatest()) {
      getWorldMesh().recreateMesh();
    }
    // do something here to change the mesh
  }
}
